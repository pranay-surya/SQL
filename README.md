# SQL for Data Analysis 

This repository contains a structured collection of **SQL concepts and hands-on practice notebooks** focused on **Data Analysis**.  
All SQL queries are practiced using **Jupyter Notebook**, allowing theory, queries, and outputs to exist in one place.

---
## Technologies Used

- SQL – Core query language
- Python – Notebook environment support
- Jupyter Notebook – Interactive execution
- MySQL – Relational databases
- IPython SQL Magic – Running SQL inside notebooks
